Everything You Need

A complete solution for LDAP directory management, designed for modern infrastructure.

User Management

Create, edit, and delete LDAP users with support for inetOrgPerson, posixAccount, and shadowAccount object classes.

Group Management

Manage POSIX groups and memberships with an intuitive interface. Add and remove members with a single click.

Sudo Roles

Full sudoers LDAP schema support. Manage sudo privileges for users and groups with fine-grained control.

SSH Key Management

Store and manage SSH public keys directly in LDAP. Support for ldapPublicKey object class integration.

LDAP Authentication

Authenticate users directly against your LDAP directory. Role-based access control with admin and read-only roles.

Audit Logging

Track all changes with detailed audit logs. Know who did what and when with complete accountability.

Dynamic Forms

Form fields automatically adapt to your LDAP schema. Support for custom object classes and attributes.

Photo Support

Upload and display user photos stored in LDAP. JPEG photo support for user profiles and avatars.

REST API

Full-featured REST API for automation. Integrate with your existing tools and workflows.

Get Started in Minutes

Deploy LDAP Warden with Docker Compose in just a few commands.

Terminal
# Clone the repository
$ git clone https://github.com/ldapwarden/ldapwarden.git
# Start all services
$ cd ldapwarden && docker compose up -d
# Access the application
$ open http://localhost:8000
# Default credentials: admin / admin123
